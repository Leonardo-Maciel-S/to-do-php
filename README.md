# To-do API

Api desenvolvida para fixar conhecimentos básicos do php, como desenvolver uma aplicação CRUD com o PHP puro e SQLite3.

## Funcionalidades

1. Listar - GET
1. Criar - POST
1. Excluir - Delete
1. Editar - PATCH

## O que aprendi

1. Sintaxe do PHP.
1. Manipulação de Bancos de Dados utilizando SQLite3.
1. PHP Server.
1. Rotas.
1. Variáveis Globais.
1. Sessions.
1. Como usar php "dentro" do HTML.
